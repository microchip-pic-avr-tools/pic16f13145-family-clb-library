# 4-Bit Binary Counter

The module has Reset and Enable inputs.

| **reset** | **enable** | **output** |
|:---------:|:----------:|:----------:|
|     0     |      0     |  Disabled  |
|     0     |      1     |   Enabled  |
|     1     |      0     |    Reset   |
|     1     |      1     |    Reset   |
